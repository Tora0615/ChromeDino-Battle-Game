Dino_Controller.cs 是以前寫的東東，可能難以看懂，
但應該可以做為後續要求的想法參考



dino 資料夾內有小恐龍動作與技能的圖

要求一 : 
利用範例code做出以下動作的動畫，並要包含操控部分
左右都要有，pygame應該有函式可以反轉左右

1. 閒置 : 
   - 大多時間睜開眼睛，偶爾眨幾下眼睛，
     可用圖片出現次數達到，不要用time.sleep，
     可能會導致程式出錯

2. 跑 : 
   - 兩張交錯，無特殊需求

3. 死亡 : 
   - 睜眼這張就好，無特殊需求

4. 蹲著(只有按住S) : 
   - 選一張當他蹲著，無特殊需求

5. 蹲著跑(同時按住S跟A或D) : 
   - 兩張交錯，無特殊需求

6. 跳 : 控制可沿用範例扣

7. 在空中按向下鍵急速落下 : 先不用

要求二 : 
只需要將技能動畫實現，以及技能的開、關
暫時可以不用管傷害、初始位置等 

1. 技能0 : 
  - 噴火&射出子彈，鍵位H，圖片名稱skill0、bullet
  - 按著開，放開關。子彈射出方向與火的方向相同

2. 技能1 : 
  - 龍捲風，鍵位J，圖片名稱skill1
  - 按下一次按鍵，召喚一個龍捲風，數秒後消失

3. 技能二
  - 打雷，鍵位K，圖片名稱skill2
  - 按下一次按鍵，召喚一次打雷，雷最大的時候短暫停留在那張，之後消失


需求三 : 
設定技能初始位置、移動軌跡，
傷害、碰撞先不用理

1. 技能0 : 
  - 噴火要在恐龍嘴邊，跟著他頭左右轉向
  - 按著技能轉向時，火的位置也要正確

2. 技能1 : 
  - 按下按鍵，龍捲風與小恐龍頭同方向射出，數秒後消失

3. 技能二
  - 按下按鍵，召喚打雷，與小恐龍頭同方向，之後消失
